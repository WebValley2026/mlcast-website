What is MLCast?

MLCast Community is a decentralized ecosystem of researchers, engineers, and meteorologists building open-source infrastructure for high-precision weather nowcasting. We leverage modern machine learning techniques to process vast amounts of radar data in real-time, providing critical lead time for severe weather events.

[MISSION]

Collaborative Research Ecosystem

Our mission is to democratize access to advanced meteorological prediction tools. By fostering an open-source community, we aim to accelerate innovation in weather nowcasting, bridging the gap between academic research and operational deployment.

  • check_circle Standardized, documented benchmark datasets for ML weather prediction.
  • check_circle Modular architecture supporting drop-in replacement of modeling architectures.
  • check_circle Transparent evaluation frameworks preventing "black box" science.

Datasets

Standardized, high-res radar archives for ML. Access data streams via our Python interface with dask-backed xarray support.

menu_book Docs Repo
Quick Access
# Install the catalog package
pip install mlcast-datasets

# Open and list available sources
import mlcast_datasets
cat = mlcast_datasets.open_catalog()
list(cat)
